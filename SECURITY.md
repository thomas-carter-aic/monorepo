# Security Policy

## Reporting a Vulnerability

If you discover a security issue:

1. **Do not create a public GitHub issue**
2. Email us at security@example.com with detailed findings
3. We will respond within 72 hours

## Supported Versions

| Version | Supported |
|---------|-----------|
| v1.x    | ✅         |
| <1.0    | ❌         |
